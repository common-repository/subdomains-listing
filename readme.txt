A multi site listing plugin.



== Description ==

- This plugin will work only in multi site wordpress site. It will display all subdomain list in admin side which you can check if need to display at site using this shortcode [list_all_subdomains].

== Installation ==

Notes : Make sure you have enabled your multi-site network in wp-config file with **define( 'WP_ALLOW_MULTISITE', true );** code.

1. Upload the plugin files to the '/wp-content/plugins/subdomain-list' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin from the 'Plugins' menu in WordPress.
